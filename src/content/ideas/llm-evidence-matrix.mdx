系统综述中需要从大量文献中提取证据构建矩阵，这个过程耗时且容易出错。尝试用大语言模型辅助文献筛选和关键信息提取，提高综述的效率和覆盖度。

## 问题背景

生态学系统综述往往需要从数十甚至数百篇文献中提取一致的证据，这个过程耗时且容易出错。大语言模型在信息提取方面表现出一定能力，值得探索。

## 初步想法

1. 使用 API 调用 GPT-4 或 Claude 进行文献信息提取
2. 构建 Prompt 模板规范提取格式
3. 设计验证流程抽检提取质量
4. 迭代优化 Prompt 提高准确性

## 风险与局限

- 幻觉问题：模型可能生成看似合理但错误的内容
- 生态学专业术语理解有限
- 提取质量依赖 Prompt 设计
- 需要人工验证环节

## 当前状态

处于初步想法阶段，尚未开始实验。

## 为什么生态学研究者要学机器学习

生态学数据通常维度高、样本量有限、空间异质性强。机器学习可以帮助我们处理非线性关系，但也需要谨慎使用。

## 从哪里开始

### 1. Python 基础

确保熟悉 pandas 和 numpy 的基本操作。

### 2. scikit-learn 入门

```python
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score
```

### 3. 常见模型

- 随机森林（分类/回归）
- 梯度提升树
- 支持向量机

## 生态学应用的常见陷阱

1. **过拟合**：样本量小时容易发生
2. **数据泄露**：训练集和测试集未正确划分
3. **生态学意义缺失**：只关注预测精度而忽略解释
